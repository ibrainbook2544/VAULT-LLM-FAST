## 一、手机上的 Obsidian 仓库存在哪里？

在 Android（三星）上：

Obsidian 的 Vault 本质上就是一个普通文件夹。

默认通常在：

```
内部存储 / Obsidian / 你的Vault名字
```

常见路径类似：

```
/Storage/emulated/0/Obsidian/YourVaultName
```

或者：

```
内部存储/Obsidian/YourVaultName
```

---

## 如何确认你的真实路径？

### 方法 1（推荐）

在 Obsidian 里：

1. 打开 Vault
2. 点击右上角「⋮」
3. 选择 **关于 / Vault 信息**
4. 查看存储路径

---

### 方法 2（更直接）

用文件管理器：

三星自带「我的文件」：

- 打开
- 进入「内部存储」
- 找到 `Obsidian` 文件夹

如果你看不到：

去检查是否创建在：

```
Documents
```

或其他目录。

---

## 二、用 Syncthing 同步到电脑（标准方案）

你需要：

- 手机安装 Syncthing
- 电脑安装 Syncthing
- 两边互相配对
- 把 Vault 文件夹设为同步目录

---

## 第一步：安装 Syncthing

### 安卓：

安装：

- Syncthing

官网：

[Syncthing 官方网站](https://syncthing.net/?utm_source=chatgpt.com)

---

### 电脑端：

去官网下载对应系统版本安装。

同样在官网：

[Syncthing 官方网站](https://syncthing.net/?utm_source=chatgpt.com)

---

## 三、具体同步步骤（一步一步）

---

## 步骤 1：手机端添加文件夹

打开 Syncthing：

1. 点击「添加文件夹」
2. 文件夹路径选择：
```
/Storage/emulated/0/Obsidian/你的Vault
```
3. 文件夹 ID 随便起名，例如：
```
obsidian_vault
```
4. 保存。

---

## 步骤 2：电脑端添加设备

在电脑 Syncthing：

- 点击「添加远程设备」
- 用手机扫描二维码
- 或输入设备 ID

完成后，两边会互相识别。

---

## 步骤 3：电脑端也添加文件夹

在电脑：

1. 添加文件夹
2. 选择你想存放 Vault 的位置，比如：
```
D:/ObsidianVault
```
3. 文件夹 ID 必须和手机一致：
```
obsidian_vault
```

这样才能自动匹配。

---

## 步骤 4：建立同步关系

在任意一端：

- 把对方设备勾选为「共享」

然后确认。

几秒后开始同步。

---

## 四、实现“按日期范围查询”

Dataview 有两种常用方式：

---

## 方式 A：基于文件名（最简单推荐）

因为你的日记文件名是：

```
YYYY-MM-DD
```

Dataview 可以直接按文件名排序或过滤。

---

### 示例 1：查询最近 7 天

在任意笔记里写：

```
LIST
FROM "Daily"
SORT file.name DESC
LIMIT 7
```

效果：

显示最近 7 篇日记。

---

### 示例 2：按日期范围查询（精确控制）

如果文件名是标准日期格式，可以这样：

```
LIST
FROM "Daily"
WHERE file.name >= "2026-05-01" 
AND file.name <= "2026-05-08"
SORT file.name ASC
```

这就是“数据库式查询”。

---

## 五、更高级方式（推荐）

## 在日记里加一个日期字段

例如在每篇日记顶部写：

```markdown
---
date: 2026-05-08
---
```

然后用：

```
TABLE date
FROM "Daily"
WHERE date >= date(2026-05-01) 
AND date <= date(2026-05-08)
SORT date ASC
```

这种方式更专业。

---

## 六、实现“自动本周 / 本月”

Dataview 支持时间函数。

例如：

### 最近 7 天：

```
LIST
FROM "Daily"
WHERE file.day >= date(today) - dur(7 days)
```

---

### 本月：

```
LIST
FROM "Daily"
WHERE file.day.month = date(today).month
```

---

## 七、如果你想更像“真正数据库”

可以升级使用：

- Dataview + Tasks 插件
- 或 DataviewJS（JavaScript 查询）

例如动态范围查询：

```
const start = dv.date("2026-05-01")
const end = dv.date("2026-05-08")

dv.list(
  dv.pages('"Daily"')
    .where(p => p.file.day >= start && p.file.day <= end)
)
```

这已经是程序级查询。
